# Load simulation data for sample size planning

**\[experimental\]** Loads pre-computed simulation results. Downloads
data if not already cached.

## Usage

``` r
load_sample_size_data()
```

## Value

A data frame with simulation results

## Examples

``` r
if (FALSE) { # \dontrun{
# Load data (downloads automatically if needed)
df_all <- load_sample_size_data()
head(df_all)
} # }
```
